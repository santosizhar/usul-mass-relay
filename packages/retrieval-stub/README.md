# Retrieval Stub

This package provides a stub implementation for retrieval that filters a sample catalog using
permission, freshness, and version criteria.

## Contents

- `src/retrieval-stub.ts`: Stub retrieval helpers using the interfaces in `packages/retrieval-interfaces`.

## Usage notes

- Sample data lives in `artifacts/examples/retrieval.documents.sample.json`.
- The stub is intended for wiring demos and validating retrieval metadata handling.
